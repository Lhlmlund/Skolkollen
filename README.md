# Skolkollen
Vi ska utveckla en hemsida med inlärning som fokus, riktad mot gymnasieval för högstadieelever.




##  Setup Instructions

### 1. Clone and install
```bash
git clone https://github.com/Lhlmlund/Skolkollen.git
cd Skolkollen

```


## Backend setup

```bash
cd backend
npm install
npm run dev


```

## Frontend setup

```bash

cd ../frontend
npm install
npm run dev

```
